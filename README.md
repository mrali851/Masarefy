# 💰 Masarefy

Masarefy is a modern Arabic Personal Finance Manager built with React + TypeScript.

## Features

- Arabic RTL UI
- Daily / Weekly / Monthly Reports
- Beautiful Dashboard
- Expense Tracking
- Income Tracking
- Budgets
- Categories
- Charts
- PWA Support
- Excel Import
- PDF Export
- Offline Support
- Dark Mode

---

## Tech Stack

- React 19
- Vite
- TypeScript
- TailwindCSS
- shadcn/ui
- Supabase
- React Query
- React Hook Form
- Zod
- Chart.js
